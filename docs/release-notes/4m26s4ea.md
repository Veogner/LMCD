# LMCD 4m26s4ea

![LMCD overview](../screenshots/lmcd-overview.png)

LMCD `4m26s4ea` focuses on performance, backup stability, and workflow polish.

## Highlights

- Faster workspace interaction and reduced UI lag during heavy server output
- Loading-state hardening for backups, diagnostics, and storage maintenance flows
- Explicit launch-settings save action for RAM, idle timeout, and port values
- Safer backup restore flow with retention protection for restore targets
- Backup retention normalized to `4` total (`2` auto + `2` manual/change snapshots)
